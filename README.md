# Памятник герою Советского Союза Г.И. Шибанову

Веб-сайт, посвященный памятнику герою Советского Союза Григорию Ивановичу Шибанову в агрогородке Комсеничи Круглянского района Могилевской области.

## Особенности

- Информация о памятнике и воинском захоронении
- Интерактивная 3D модель памятника с возможностью вращения и масштабирования
- Адаптивный дизайн для всех устройств
- Готов к размещению на GitHub Pages

## Структура проекта

```
otez-project/
├── index.html          # Главная страница
├── styles.css          # Стили CSS
├── model-viewer.js     # JavaScript для работы с 3D моделью
├── model.glb           # 3D модель памятника
└── README.md          # Этот файл
```

## Как разместить на GitHub Pages

### Пошаговая инструкция:

1. **Создайте новый репозиторий на GitHub**

   - Название репозитория может быть любым (например: `monument-shibanov` или `памятник-шибанов`)
   - Можете сделать его публичным или приватным (для приватных нужен GitHub Pro)

2. **Загрузите все файлы проекта в репозиторий**

   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ваш-username/название-репозитория.git
   git push -u origin main
   ```

   Или используйте GitHub Desktop / веб-интерфейс GitHub для загрузки файлов

3. **Включите GitHub Pages**

   - Перейдите в репозиторий на GitHub
   - Нажмите **Settings** (Настройки)
   - В левом меню найдите **Pages**
   - В разделе **Source** выберите ветку `main` (или `master`)
   - В папке выберите `/ (root)` (корень)
   - Нажмите **Save**

4. **Дождитесь публикации**

   - GitHub обработает ваш сайт (обычно 1-2 минуты)
   - Вы увидите сообщение: "Your site is live at..."

5. **Ваш сайт будет доступен по адресу:**
   ```
   https://ваш-username.github.io/название-репозитория/
   ```

### ✅ Что будет работать на GitHub Pages:

- ✅ Все HTML, CSS и JavaScript файлы
- ✅ 3D модель (файл model.glb) - загрузится автоматически
- ✅ Вращение и масштабирование 3D модели
- ✅ Все стили и анимации
- ✅ Адаптивный дизайн

### ⚠️ Важные замечания:

- **Размер файла model.glb**: Если файл больше 100MB, GitHub может не принять его. В этом случае используйте Git LFS или разместите модель на внешнем хостинге.
- **Время загрузки**: Большие 3D модели могут загружаться медленно. Это нормально.
- **HTTPS**: GitHub Pages автоматически использует HTTPS, что решает все проблемы с CORS.

## Локальный запуск

**⚠️ ВАЖНО:** Для корректной работы 3D модели **обязательно** необходимо использовать локальный веб-сервер! Открытие файла напрямую через `file://` не будет работать из-за политики безопасности браузера (CORS).

### Быстрый запуск

Используйте готовый скрипт:

```bash
./start-server.sh
```

Или вручную:

```bash
# Python 3
python3 -m http.server 8000

# Или Python 2
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере: **http://localhost:8000**

### Альтернативные способы запуска сервера

**Node.js (если установлен):**

```bash
npx http-server -p 8000
```

**PHP (если установлен):**

```bash
php -S localhost:8000
```

## Управление 3D моделью

- **Вращение**: Перетаскивание мышью
- **Масштабирование**: Колесико мыши
- **Перемещение**: Зажатая средняя кнопка мыши (или Ctrl + перетаскивание)
- **Сброс вида**: Кнопка "Сбросить вид"
- **Автоповорот**: Кнопка "Автоповорот"
